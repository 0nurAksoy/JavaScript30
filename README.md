## 📖 About

Each project is built using only vanilla JavaScript — no jQuery, no frameworks. The goal was to strengthen core JS fundamentals including DOM manipulation, events, arrays, timers, APIs and more.

Original challenge by [Wes Bos](https://wesbos.com).
